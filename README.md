Flask 全栈开发个人项目 异地换宿 (Switch Home)
===


体验
---
- url: https://www.chenyehong.com
- 无需登录即为游客
- 动图加载比较慢

主要功能
---
- 用户模块
- 板块:
    - 寄宿信息平台:
        - 为本地大学生提供外地在校儿女的空闲卧室
    - 交流社区: 
        - 父母社区
        - 学生社区
        - (增进"大学生"家庭成员之间相互相互了解)
- 根据发布时间或被回复时间排序
- 支持标签检索:
    - 根据 [大学 / 城市] 分类检索 [寄宿信息 / 话题]
- 邮件提醒:
    - 私信, 修改密码
- 话题编辑:
    - Markdown 语法实时预览
    - 用户图片动态上传 & 自动填写链接
    - 封面动态上传 & 自动生成缩略图
